## 🎯 KAYIT REHBERİ:

1. **Her dosyayı ayrı ayrı kaydet** - Tek seferde değil

4. **Dosya adlarını tam olarak** yukarıdaki gibi yap
5. **Her karakter farklı ses tonu** kullan
